# 📘 Assignment: Games in Python

## 🎯 Objective

Build a classic Hangman word-guessing game in Python, practicing string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Implement the Hangman Game

#### Description
Create a Hangman game where the player guesses letters to reveal a hidden word before running out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the player and show current progress in `_ _ _` format
- Track the number of incorrect guesses remaining
- End the game when the word is fully guessed or all attempts are exhausted
- Display an appropriate win or lose message at the end
